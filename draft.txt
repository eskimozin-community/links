Funcionamento da funcionalidade de exibir campanhas
- Terá um array de campanhas
  - Dados das campanhas:
    - Título
    - Subtítulo
    - Link
    - Nome do link para exibição
    - Data de início
    - Data de fim
- Deve ser verificado num Código se há alguma campanha válida
  - A campanha é válida se a data atual for maior ou igual à data de início
  - E se a data atual for menor ou igual à data de fim
  - Para ser válida deve ter todas as informações necessárias (descritas acima)
- Se houver uma ou mais campanhas válidas, devem ser exibidas em ordem de finalização (a que está encerrar primeiro deve ser mostrada antes)
